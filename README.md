# ProyectoFinal
## Descripción del Problema
¿Cómo podemos calcular el costo de un seguro médico basándonos en variables como la región, sexo, índice de masa corporal (IMC), número de hijos, hábito de fumar y edad? Nuestro análisis se centrará en identificar cuáles de estas variables tienen mayor influencia en el incremento del costo del seguro. El objetivo es desarrollar un modelo predictivo que no solo estime los costos de manera eficiente, sino que también determine los factores clave que contribuyen al aumento de estos costos.
